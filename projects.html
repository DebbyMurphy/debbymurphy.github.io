HTML & CSS

In the first session of the 6 month SHE{CODES} Plus journey we launched straight into HTML & CSS with no time to waste.

Through a variety of guided and self paced exercises I learnt to write semantic HTML for readability and accessibility, use developer tools for debugging, and implement responsive design.

HTML, CSS and some basic JS was used to build this portfolio site, and was used to style views, templates and components in three Django/React sites.

Python & JSON

Learning to work with Python and JSON data was possibly the most mind blowing and rewarding aspect of the whole course.

I learnt how to use PowerShell, virtual environments, and worked with:

  • Variables, Booleans, Floats, Integers, Strings
  • Lists, Dictionaries
  • If Statements, Functions
  • For Loops, While Loops
  • The Plotly Library

Django & React

Using Django I built a news website allowing authors to sign up and post articles. Readers are able to browse articles by particular authors, or simply view the latest news.

This website required simple relational databased schema, as well as user authentication.

Meeting Russel Keith-Magee and hear him talk about User Authentication, really engrained into me the importance of building for users. In most cases, asking for a first name and last name is totally uncessary, and sometimes offensive.
